Cabeçalhos H1 até H6

Os cabeçalhos servem para criar títulos e subtítulos na página. O h1 é o título principal, sendo o tamanho maior e geralmente usado para títulos e seções mais importantes. Eles não servem só para mudar o tamanho, mas também ajudam o Google e os leitores de tela a entenderem a estrutura da página, tipo um sumário. Os diferentes tamanhos estão no exemplo abaixo:

Parágrafos

Os parágrafos são feitos com a tag <p>. Tudo que vamos escrever em texto normal vai dentro dela. O navegador automaticamente dá um espaço antes e depois do parágrafo, deixando o texto bem organizado e fácil de ler.

Listas ordenadas e não ordenadas

Existem dois tipos principais de lista:

  • Lista ordenada: Criada com a tag <ol>.
    Coloca números automaticamente. Perfeita quando a ordem importa, tipo um passo a passo.

  • Lista não ordenada: Criada com a tag <ul>.
    Usa marcadores como bolinhas ou quadradinhos. É boa para listas simples, tipo ingredientes de uma receita.

Em ambas, cada item da lista é feito com a tag <li>.

Links

O link é feito com a tag <a>. Seguido pelo atributo `href`, onde colocamos o endereço do site para onde o link vai.

Ao clicar no texto, vamos para o site do Google. Permitindo a navegação dentro do seu próprio site ou para páginas externas.

Tipos de links

Links externos:

São aqueles que levam para outro site, devemos colocar o endereço completo da página para onde o link vai. Como no exemplo acima.

Links internos (navegação dentro da própria página):

A navegação interna de um site é chamada de ancoragem. Colocamos um identificador (id) em alguma parte da página e criamos um link apontando para ele usando o símbolo de hashtag `#`.
Exemplo: Ir para a seção Sobre.
Ao clicar, a página rola sozinha até aquela parte, que deve estar com o id igual ao de seu link.
É muito útil em páginas longas, tipo currículos ou artigos grandes.

Link para e-mail:

Dentro do atributo href colocamos o comando “mailto:” seguido do endereço de e-mail após os dois pontos, ao clicar abrimos o programa de e-mail padrão. Enviar e-mail

Link para telefone:

Dentro do atributo href colocamos o comando “tel:” seguido pelo número de telefone após os dois pontos, geralmente usado em seções de contatos. Exemplo: Ligar agora

Abrir link em uma nova aba:

Para não fechar a página atual quando a pessoa clica, adicionamos o atributo target="_blank".
Exemplo: Abrir Google em nova aba
Geralmente é usado quando o link direciona para outro site e para o usuário não sair da página atual ao clicar em algum link.

Atributos

Atributos são informações extras colocadas nas tags HTML para dar mais detalhes. Eles sempre vão no formato `nome="valor"`.
Por exemplo, no link usamos `href="url"`, em uma imagem usamos `src="foto.jpg"` e `alt="descrição"`.
Os atributos mudam o comportamento ou dão mais informações para uma tag.
